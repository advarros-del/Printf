DESCRIPTION

In this proyet our purpose is emulate the principal imputs: 'cspdiuxX%' of the original function printf.
This function print something.

INSTRUCTION

If you want to use the function you have to write the character '%' and imidietly after the
type of the character/s that you want or you expect. These character/s could be:
·'c' to print only one character.
·'s' to print a string.
·'p' to print a pointer.
·'d' to print one or more digits.
·'i' to print one or more digits.
·'u' to print an unsigned int. 
·'x' to print a hexadecimal table with the 'abcdef' in lowercase. 
·'X' to print a hexadecimal table with the 'ABCDEF' in capital letters.
·'%' to print a the character '%'.

If don't specify which type of character is, you will going to recieved an string.
If the string empty you will not going to print nothing.
This function returns alwais the total imputs ('%') that you give them.
